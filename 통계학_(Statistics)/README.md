## 📌 Focus
- **노이즈/이상치 제어**: 로봇 센서(LiDAR, IMU, 등)에서 발생하는 이상치들을 통계적으로 판별하고 제거하여 데이터의 신뢰성을 확보
- **데이터 정규화**: 서로 다른 단위를 가진 센서 데이터를 정규화하여 로봇 제어 알고리즘이나 학습 모델의 연산 효율을 최적화
